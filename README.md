# ZIMO ZCAN NPM Module

Typescript library for the ZIMO ZCAN v2 protocol via UDP. 

It is also available as a node module at npmjs.com.

## Getting Started

Simply add the library to your npm based project by running 

`npm install @zimo-elektronik/zcan` 

## Further Information

See the ZCAN v2 documentation at https://www.zimo.at/web2010/documents/Zimo_CAN-Protokoll_4.35.pdf
